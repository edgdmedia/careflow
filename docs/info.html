End-to-End Flow
How a participant moves through CareFlow
1
Intake Agent

Free-text inquiry arrives

Someone texts "I've been struggling with anxiety, can't sleep" via WhatsApp. The Intake Agent parses the message using Qwen NLP, identifies presenting concern, and begins a structured conversation to collect name, availability, and session preference — without asking a form-style questionnaire.

2
Eligibility Agent

Screening & matching

The Eligibility Agent checks the participant against group criteria (e.g., anxiety focus, age range, no active crisis). Straightforward matches proceed automatically. Edge cases are flagged to the therapist dashboard for manual review — the human-in-the-loop checkpoint.

3
Scheduling Agent

Cohort assignment & confirmation

Once approved, the Scheduling Agent checks available cohort slots, assigns the participant, updates the calendar, and sends a confirmation message with onboarding information — all within seconds of therapist approval.

4
Pre-Session Agent

24-hour preparation sequence

The night before each session, the Pre-Session Agent sends a reminder, a short check-in ("How are you feeling today on a scale of 1–10?"), and any preparatory materials. Responses are logged and surfaced to the therapist before the session starts.

5
Post-Session Agent

Automated session close

After each session, the Post-Session Agent sends a feedback form, delivers the week's worksheet, marks attendance, and logs the session to the therapist's training hours tracker — creating a full audit trail for licensing compliance.

Short Description (for Devpost tagline)

↳ use in the one-line description field

CareFlow is an autonomous AI agent that handles the full therapy intake and session management workflow — from first message to session log — so therapists can focus entirely on their clients.

Inspiration

Mental health therapists spend a significant portion of their working time on administrative work that never helps a single client: chasing intake forms, manually scheduling cohort groups, sending reminders, collecting session feedback, and updating training logs for licensing compliance. For therapists in training — who need to document every client hour to meet licensing requirements — this burden is especially acute.

We built CareFlow because we experienced this problem firsthand. Running a group therapy program called Anxiety Unplugged — a six-week anxiety recovery series — we found that the logistics of intake, scheduling, and post-session follow-up consumed hours that should have gone toward preparation and presence. CareFlow automates that entire operational layer, so a therapist can take on more clients, document hours accurately, and deliver better care without drowning in admin.

What It Does

CareFlow is a multi-step autonomous agent pipeline that manages the complete pre- and post-session workflow for group or individual therapists:

  • Intake Agent: Receives free-text inquiries via WhatsApp, web form, or email. Uses Qwen NLP to extract presenting concern, name, and availability through a natural conversation — no rigid forms.
  • Eligibility Agent: Screens participants against group therapy criteria. Flags edge cases for therapist review before any scheduling occurs (human-in-the-loop checkpoint).
  • Scheduling Agent: Assigns participants to available cohort slots, sends confirmation messages with onboarding info, and updates the therapist's calendar automatically.
  • Pre-Session Agent: Sends reminders and short check-in questions 24 hours before each session. Surfaces participant mood data to the therapist before the session starts.
  • Post-Session Agent: Delivers session worksheets, collects feedback, logs attendance, and updates the therapist's training hours tracker for licensing compliance.
  • Therapist Dashboard: A clean Next.js interface showing participant progress, cohort overview, session logs, and training hours — with all data populated by the agents, never manually entered.

How We Built It

CareFlow is built on Qwen Cloud infrastructure, with Qwen-Max powering the NLP reasoning across all five agent steps. The backend runs on Alibaba Cloud ECS, hosting a self-deployed n8n instance that orchestrates the agent workflow — routing messages between agents, managing state, handling retries, and triggering the human-in-the-loop approval checkpoint.

Participant data and session logs are stored in PostgreSQL via Supabase. Communication with participants happens via the WhatsApp Business API and SMTP email. The therapist dashboard and intake form are built with Next.js. Calendar integration uses the Google Calendar API for slot management and automated booking.

Each agent in the pipeline is a discrete Qwen API call with structured tool use — designed so that any single agent can be paused, overridden, or extended without rebuilding the pipeline. This modularity means CareFlow can be adapted to individual therapy, peer support groups, or institutional mental health services.

Challenges We Ran Into

  • Ambiguous free-text intake: Real inquiries are messy — people describe mental health concerns in personal, sometimes vague language. Prompt engineering the Intake Agent to extract structured data without being clinical or cold required significant iteration.
  • Eligibility edge cases: Some participants present with concerns that are adjacent to the group's focus but not a clean match. Building an agent that reasons about these cases probabilistically — and knows when to escalate — was the most technically nuanced part of the build.
  • State management across sessions: Because the workflow spans days (intake → session 1 → session 2 → session 6), maintaining coherent participant state across n8n runs required careful schema design and idempotent agent logic.

Accomplishments We're Proud Of

  • A fully end-to-end pipeline from raw WhatsApp message to confirmed session booking, requiring zero therapist input for standard cases.
  • A human-in-the-loop checkpoint architecture that keeps the therapist in control of clinical decisions without burdening them with operational ones.
  • A training hours tracker that automatically generates the documentation therapists-in-training need for licensing — a genuine time-saver with real professional stakes.
  • A culturally-grounded intake experience: the agent's conversational style was calibrated for warmth and accessibility, not clinical distance — reflecting the values of the Unclutter mental wellness platform it's built for.

What We Learned

Building CareFlow taught us that the hardest part of agentic AI for sensitive domains isn't the AI capability — it's the design of escalation. Knowing when an agent should proceed autonomously and when it must pause for a human is a product decision as much as a technical one. Getting that boundary right in a mental health context required us to think carefully about risk, consent, and clinical responsibility in ways that generic automation frameworks don't account for.

We also learned how powerful Qwen's reasoning models are for structured extraction from conversational text — the quality of intake parsing significantly exceeded our expectations.

What's Next for CareFlow

  • Integration with the full Unclutter app suite as the operational backbone for all group therapy programs
  • Voice intake using Qwen's multimodal capabilities — so participants can leave a voice note describing their situation rather than typing
  • Outcome tracking — longitudinal mood and progress data surfaced to therapists across a participant's full program journey
  • An open-source release of the core agent pipeline, enabling other therapists and mental health organisations to deploy CareFlow for their own programs
